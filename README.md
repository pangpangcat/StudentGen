# StudentGen

# About the project
This is a student service application.

# Technology used
- Java
- Object Oriented Programming

# Feature
  * Create a new student.
  * Enrroll the student to few courses.
  * Grade the student.
  * Show the students and courses summary and verify that data is correct.
